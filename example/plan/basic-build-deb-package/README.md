
# demo plan for make deb package


## build deb

``` sh
$ make deb-build
```


## install deb

``` sh
$ make deb-install
```


## remove deb

``` sh
$ make deb-remove
```


## clean deb

``` sh
$ make deb-clean
```


## help

``` sh
$ make
```

or

``` sh
$ make help
```


## serve

``` sh
$ make serve
```
